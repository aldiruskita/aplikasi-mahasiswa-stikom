Aplikasi Android untuk mengelola data siswa menggunakan database SQLite.
Aplikasi ini dibuat untuk memudahkan pengguna dalam melakukan pengelolaan data siswa secara digital dengan tampilan yang sederhana, menarik, dan mudah digunakan.

✨ Fitur Utama
🗄️ Kelola Data dengan SQLite
Menggunakan database SQLite untuk penyimpanan data secara permanen
Mendukung operasi CRUD:
Create → Menambah data siswa
Read → Menampilkan data siswa
Update → Mengubah data siswa
Delete → Menghapus data siswa
Struktur database tersusun dengan baik dan efisien
📋 Menampilkan Data di List
Menggunakan RecyclerView untuk menampilkan daftar siswa
Desain modern menggunakan CardView
Menampilkan informasi:
Nama Siswa
NIS
Kelas
Rata-rata Nilai
🎨 Indikator Warna Nilai
Warna	Keterangan	Nilai
🟢 Hijau	Sangat Baik	≥ 80
🟠 Orange	Baik	60 - 79
🔴 Merah	Perlu Perbaikan	< 60
📊 Menyimpan Nilai dalam Database
Nilai siswa tersimpan secara permanen di SQLite
Perhitungan rata-rata nilai dilakukan otomatis
Validasi input nilai:
Minimal: 0
Maksimal: 100
